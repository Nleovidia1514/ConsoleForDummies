CD      [route]\[..]            MOVE THROUGH SYSTEM DIRECTORIES
ECHO    [messagee] || [>][name] SHOWS MESSAGES ON CONSOLE || CREATES A TXT FILE WITH SPECIFIED MESSAGE
DIR     [NO ARGUMENT]           SHOWS A DETAILED LIST OF ALL THE FILES IN CURRENT DIRECTORY
COPY    [file1][file2]          COPIES A FILE FROM SOURCE TO DESTINATION
DEL     [file]\[dir]            DELETES A FILE OR A DIRECTORY
MOVE    [file][dir]             RENAMES OR MOVES A FILE FROM ITS CURRENT POSITION TO THE SPECIFIED DIRECTORY
MKDIR   [dir]                   CREATES A NEW DIRECTORY
DATE    [NO ARGUMENT]           SHOWS SYSTEM DATE AND TIME
