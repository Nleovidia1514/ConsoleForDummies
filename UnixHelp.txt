CD      [route]\[..]            MOVE THROUGH SYSTEM DIRECTORIES
ECHO    [message] || [>][name]  SHOWS MESSAGES ON CONSOLE || CREATES A TXT FILE WITH SPECIFIED MESSAGE
LS      [void]                  SHOWS A DETAILED LIST OF ALL THE FILES IN CURRENT DIRECTORY
CP      [file1][file2]          COPIES A FILE FROM SOURCE TO DESTINATION
RM      [file]\[dir]            DELETES A FILE OR A DIRECTORY
MV      [file][dir]             RENAMES OR MOVES A FILE FROM ITS CURRENT POSITION TO THE SPECIFIED DIRECTORY
MKDIR   [dir]                   CREATES A NEW DIRECTORY WITH THE SPECIFIED NAME
DATE    [NO ARGUMENT]           SHOWS THE SYSTEM DATE AND TIME
